# contact-form

# アプリケーション名
contact-form

## 環境構築
1.git clone https://github.com/momoca-13/contact-form.git

2.docker-compose up -d --build  

3.docker-compose exec php bash　

4.php artisan migrate  

5.php artisan db:seed

## 使用技術（実行環境）
PHP 7.4.9  

Laravel  8.83.8  

mysql 8.0.26
